About

Jackaroo is a small Java game created to practice OOP, basic game loop logic, keyboard input handling and simple 2D rendering using the Java standard libraries (or a lightweight library if you prefer). The goal of the project is to demonstrate clean code structure, modularity, and an approachable gameplay loop for beginners.

Features

Simple, responsive 2D gameplay loop

Keyboard controls (move, jump, action)

Basic enemy / obstacle logic

Score and level progression

Configurable difficulty (via constants/config file)

Easy-to-read code intended for learning
